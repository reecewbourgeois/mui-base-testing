## Getting Started

### Prerequisites
- [NodeJS](https://nodejs.org/en)
  - (for Windows) I recommend installing NodeJS using this tool: [NVS](https://github.com/jasongin/nvs)

### Starting the Server
- Open a terminal
- Run `npm start`
